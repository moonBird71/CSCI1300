My CIST 1300 Assignments

Loki Main Page

The main page for the loki server that the students of CIST 1300 (and CSCI 2850) use to practice classwork. The current assignment, a7, requires this link to be included.

Assignment 1

The first assignment was just to set up a web space. Professor Blackmore had us put a temporary home page up to fill it. The temp page had three random things with a vaildator badge at the bottom.

Assignment 2

This assignment was to create an actual webpage and incorporate images. We got rid of the three random things and replaced them with personal information, pictures and several quotes.

Assignment 3

In this assignment, we were to create a table containing our daily schedule on a new page. Then we linked that page to our classes' webpages, our favorite sites, and to the webpage we made in our previous assignment. So, it looked exactly like assignment 2 except for a link at the bottom: the link then would open up to the rest of assignment 3.

Assignment 4

Assignment 4 started us on CSS styling. We added on to our previous assignments by incorporating music. We made yet another page and used inline styling to embed videos and music from Youtube and archive.org. This page was also linked to the other two pages through linis at the bottom of the pages.

Assignment 5

This assignment took us further into CSS. We deleted a great deal of the inline styling from the previous assignment in order to apply external styling. We created an external file that held the styling rules we wanted to be applied to the webpages we already made in the previous assignments.
